Project Two

"Resposnisve Club Website"

The 2nd "off platform" HTML/CSS project let me further practice my freedom of layout and design, I made a serious attempt at keeping the styling within a colour scheme, and using images that represent my art, I really took care to make sure the layout remains smooth and responsive to browser screen size.

Unlike using "flex" css rules, the "@media and screen" rules let me specifically controll styles and flow of content to screen size. removing or changing elements as needed.

With this front-end project I really found time to like how my styling represented my content, helped me more appriciate working with the code ive been learning.
